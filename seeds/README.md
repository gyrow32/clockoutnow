# Seeds Folder

Drop any seed here — a note, a link, a screenshot, an idea.
Name the file anything: `retell-voice.md`, `cool-tool.txt`, `idea.md`

Chrissy's Site Improver checks this folder every 15 min.
When it finds a new seed, it researches, plans, and adds to the backlog.

## Format (optional)
```
# Seed: [Name]
**What:** Brief description
**Why it matters:** How it helps ClockOutNow
**Links/details:** Any URLs, account info, etc.
```

Or just throw in raw text. I'll figure it out.
