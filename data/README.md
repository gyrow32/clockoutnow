# data/

This folder is for **local-only inputs/outputs** (often contains PII):

- Imports (spreadsheets, raw lead dumps)
- Script outputs (scraped leads JSON, exports)
- Legacy artifacts (SQLite databases, one-off task files)

By default, `data/` contents are **gitignored** (see `.gitignore`). Keep only non-sensitive examples here if you ever need to commit something.
